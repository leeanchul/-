# 롯데칠성글로벌ERP 프로젝트

1) 프로젝트명: 롯데 칠성 음료 글로벌 ERP 프로젝트

- 소속회사 : 투비웨이
- 수행 기간 : 2025.05 ~ 2026.01 (약 9 개월)

- 주요 역할 : ERP 고도화 솔루션 백엔드 개발 및 데이터 파이프라인 운영·최적화

- 업무 성과 :
1. 배치 성능 최적화
- Quartz 기반 배치 Job 병목 구간 분석 및 개선
- 실행 계획 기반 쿼리 튜닝 수행
- [성과] 조회 속도 15초 1초 미만 (93% 개선)

2. SAP 데이터 ETL 파이프라인 구축
- SAP ERP 데이터 서비스 DB 이관 프로세스 구축
- Apache Hop 기반 ETL 자동화 구현
- [성과] 수작업 제거를 통한 업무 효율화 및 데이터 이관 안정성 확보

3. 데이터 정합성 및 비즈니스 로직 개발
- MBR 검사 로직 및 Assignment 자동화 기능 개발
- JSON 기반 유연한 데이터 구조 설계
- [성과] 데이터 입력 오류 감소 및 시스템 무결성 확보

4. 장애 분석 및 데이터 검증
- 데이터 오류 원인 분석
- DB 메타 정보 기반 전수 조사 수행
-[성과] 데이터 비정상 적재 지점 포착 및 잠재 장애 요인 사전 제거

5. 외부 연계 API 개발 및 운영
- 주소/연락처 API 개발 및 유지 보수
- 장애 발생 시 로그 분석 및 데이터 보정 수행
-[성과] 예외 상황 신속 조치를 통한 서비스 연속성 및 안정성 유지
